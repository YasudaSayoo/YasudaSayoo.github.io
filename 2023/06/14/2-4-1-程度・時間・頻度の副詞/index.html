2-4-1 副詞＿＿程度・時間・頻度の副詞

程度・時間・頻度の副詞

言葉と例文

ウォーミングアップ

上司为什么生气？——

部下：部長はなかなかカラオケがお上手ですね。
上司：何でそんなに偉そうなんだ。お前だって、そんなに上手じゃないだろ！

言葉

  • 程度（強調）
    • 実に　彼女の発表は実にすばらしかった。
    • 極めて　その計画には極めて大きな問題がある。
    • よほど・よっぽど　彼はよほど怒っていたのか、挨拶もしなかった。
    • 大いに　昨日は部下たちと大いに飲んだ。
  • 程度（比較）
    • 余計（に）　彼女は、人より余計に努力して、選手に選ばれた。
    • 何より　幸せに生きていくためには、健康が何より大切だ。
    • はるかに　バスより電車の方がはるかに早く着くよ。
  • 程度（少なく）
    • 若干　前に会ったときより、彼女は若干太ったようだった。
    • いくらか　今日は昨日よりいくらか暖かかった。
  • 程度（変化）
    • 一層　今週は先週より一層寒くなった。
    • 一段と　彼は留学して、一段と日本語が上手になった。
    • なお・なおさら　会えないと聞いて、なおさら会いたくなった。
    • ぐっと　給料が３万円上がれば、生活もくっと楽になるのだが。
    • めっきり（と）　最近、お客の数がめっきり減った。
  • 程度（基準以上）
    • 割合・割と　今年の冬は割合暖かい。
    • 相当　彼女のお父さんは相当怖いらしいよ。
    • なかなか　あの映画はなかなか面白かったよ。
  • 時間（瞬時）
    • 直ちに　建物が倒れる危険がありますので、直ちにこの建物より出てください。
    • たちまち　テストが渡されると、彼はたちまち問題を解いてしまった。
  • 時間（過去）
    • あらかじめ　うちに来る時は、あらかじめ電話をしてもらえますか。
    • いつの間に（か）　さっきここにあったケーキがいつの間にかなくなっていた。
    • もはや　借金ばかりで、うちの会社ももはや終わりだ。
    • 従来　この病気は、従来は手術をしていたが、今は薬で治る。
  • 時間（未来）
    • いずれ　あんなに仲がいいのだから、二人はいずれ結婚するに違いない。
    • 今に　そんな運転の仕方では、今に事故を起こすよ。
    • もはや　そのうち怒っていたことも忘れるでしょう。
    • 従来　日本では近々選挙が行われるそうだ。
  • 時間（一時）
    • ひとまず　血が止まらなかったので、ひとまず近所の医者に行くことにした。
    • いったん　忘れ物をしたので、いったんうちに戻った。
  • 時間（その他）
    • 依然（として）　アジアの国々は好景気だが、日本では依然として不景気が続いている。
    • 相変わらず　彼は相変わらずカラオケによく行っている。
    • 絶えず　彼女は周りの人に絶えず愚痴を言っている。
    • 終始　結婚式の間、彼は終始笑顔だった。
    • 当分　猛暑はまだ当分続くという。
    • 長年　彼女は長年この近所に住んでいる。
  • 頻度
    • 時折　風でドアが時折がたがたと鳴った。
    • まれに　その植物はまれに花を咲かせることがある。
    • たびたび　授業中、先生にたびたび注意された。
    • しょっちゅう　うちの子はしょっちゅうけがをしている。
    • しきりに　店のものを買ってほしいらしく、店員がしきりに私に話しかけてきた。

基本練習

用法

下線部に注意して、□のなかから適当な言葉を選んで、（　）に入れなさい。

  • よほど　　大いに　　ぐっと
    • こんな高い車が買えるなんて、彼は（ よほど ）稼いでいるにちがいない。
    • 就職してから、彼は（ ぐっと ）大人っぽくなった。
    • 今日はパーティーを（ 大いに ）楽しんでいってください。
  • 絶えず　　近々　　いつの間に
    • ここにあったコップは（ いつの間に ）なくなったんだろうか。
    • 彼女は自分の髪型を気にして、（ 絶えず ）鏡を見ている。
    • その駅にも（ 近々 ）エレベーターが設置される予定。
  • 依然　　終始　　当分
    • 我がチームは、試合中（ 終始 ）攻め続けた、結局、一点もとることができなかった。
    • 体を壊したので、（ 当分 ）の間、店を休みます。
    • 対策が取られたが、少年による凶悪な犯罪は（ 依然 ）減っていない。

意味

□の中から適当な言葉を選んで、（　）に入れなさい。

  • 何より　　余計に　　はるかに
    • A国の自動車の輸出台数は、輸入台数を（ はるかに ）上回っている。
    • お客さんが来るので、ビールを少し（ 余計に ）買っておいた。
    • 帰国していいことはいろいろあったが、母に会えたことが（ 何より ）嬉しかった。
  • めっきり　　なおさら　　一段と
    • 展示されていたどの作品もすばらしかったが、彼の作品は（ 一段と ）すばらしかった。
    • 最近、公園で遊ぶ子供が（ めっきり ）減ってしまった。
    • 悪いと知っていてやったのなら、（ なおさら ）よくない。
  • 相当　　極めて　　なかなか
    • 嘘だとわかっていても、褒められると、（ なかなか ）うれしいものだ。
    • 飲酒運転は、（ 極めて ）危険な行為であり、許されるものではない。
    • 昨日の夜は（ 相当 ）お酒を飲んでしまったので、今日は朝から気持ちが悪い。
  • たびたび　　まれに　　しきりに
    • 手術が必要な病気だが、（ まれに ）手術をしなくても、治る場合がある。
    • お忙しいところ（ たびたび ）お越しいただいて、申し訳ありません。
    • さっきから、こっちを見て（ しきりに ）手を振っているけど、なんか用でもあるのかな。
  • 相変わらず　　長年　　絶えず
    • 北海道は（ 長年 ）親しんだ土地なので、思い出がたくさんあります。
    • 彼は、もう三十歳になったというのに、（ 相変わらず ）仕事もせず、ゲームばかりしている。
    • 自分のやり方が間違っていないか、（ 絶えず ）見直しをすることが大切だ。
  • もはや　　あらかじめ　　従来
    • がんになった後でタバコをやめても、（ もはや ）手遅れだ。
    • 5名以上でご来店いただく場合は、（ あらかじめ ）ご連絡ください。
    • （ 従来 ）三時間かかっていた作業が、一時間でできるようになった。

用法

下線の言葉を使い方が正しい分には⭕️、間違っている文には❌を（　）に入れなさい。
また、間違っている場合には、下線の言葉に代わる正しい言葉を書きなさい。

  • 少し遅れるという連絡があったから、近々来るでしょう。 ( ❌ )  もうじき 
  • 野菜は、煮えたら、いずれ取り出して、置いておきます。 ( ❌ )  いったん・ひとまず 
  • 病気ををしたせか、父はよほど痩せてしまった。 ( ❌ )  めっきり・ずいぶん・とても 
  • 大学を卒業してからも、友人から時折連絡が来る。 ( ⭕️ )   
  • 全米で大ヒットした映画が近々公開されるそうだ。 ( ⭕️ )   
  • 人というものは、たびたび自分の失敗を人のせいにする。 ( ❌ )  しばしば 
  • 彼は、お酒を飲むと、たちまち顔が赤くなります。 ( ⭕️ )   

用法

下線の部分を変えて、正しい文にしなさい。

  • 今日食べた料理は、なかなかおいしくなかったね。 → おいしかった 
  • 彼のことをよく知って、なおさら好きです。 → 好きになりました 
  • 彼は態度が悪いから、今に友達がいなくなったよ。 → いなくなる 
  • 不景気のせいで、店の売り上げが若干激減した。 → 減少した・減った 

実践練習

    • いずれ

    • いまに
    • ただちに
  • 彼の引退コンサートは、（ １ ）感動的なものだった。

2.__の言葉に意味が最も近いものを、１・２・３・４から一つ選びなさい。

  • あの人は暇なのか、用事もないのに、しょっちゅう家に来る。（ ３ ）
    • 時折
    • たまに
    • 頻繁に
    • ときどき
  • 彼の態度がよほど許せなかったのか、彼女は怒って出ていってしまった。（ １ ）
    • 相当
    • 若干
    • はるかに
    • ちょっと
  • 三日前から風邪を引いているが、今日はいくらか体が楽になった。（ ３ ）
    • とっても
    • だんだん
    • 少し
    • 急に
  • 社会は絶えず変化している。（ ３ ）
    • だんだん
    • ゆっくり
    • いつも
    • 少しずつ
  • 祖父の病状は極めて深刻で、親戚が皆見舞いに訪れた。（ ４ ）
    • 大いに
    • 一層
    • 最も
    • 非常に

3.次の言葉の使い方として最も良いものを、１・２・３・４から一つ選びなさい。

  • 相当 （ ３ ）
    • あの人は、山田さんと相当遊ばなかったそうですよ。
    • あの人は、年に一回という相当な割合で恋人と会っている。
    • 環境問題についての本は多いので、読むのに相当時間がかかるだろう。
    • あの人は、相当日本人だ。
  • いまに （ ４ ）
    • うちの犬は、玄関を開けるといまに走ってくるんですよ。
    • 一生懸命練習していたから、いまに料理がとても上手になった。
    • 彼は、いまにずっとサッカーを続けてきた。
    • もっと注意深くやらないと、いまに失敗するよ。
  • いったん （ １ ）
    • この強風だと、いったん戻らないと危ない。
    • 今度、いったん遊びに行きませんか。
    • この本はいったん最後まで読み終えました。
    • パリにはいったん新婚旅行で行ったことがあります。
  • たちまひ （ ３ ）
    • この小説は、読んでいるうちにたちまち面白くなってきた。
    • このパソコンは一回スイッチを切ってしまうと、たちまち使えるようになるまで時間がかかる。
    • 妹はあれほど汚れていた部屋をたちまちきれいにしてしまった。
    • あの歌手は、アルバムを出すたびに、たちまち歌がうまくなった。
  • 割合 （ ２ ）
    • 中学生になってkら、割合急激に背が伸びた。
    • この映画は、割合面白かったけど、絶賛するほどではなかった。
    • 今週になって、急に真冬のように割合寒くなった。
    • 昨年の記録的な暑さと比べても、割合
答え：

  • 「相当」表示“高出一般的程度”。1句和4句中在「相当」的后面没有接表示程度的词，因此不能使用「相当」。2句表示程度低，也不能用「相当」。
  • 「今に」意为「近い将来」。
