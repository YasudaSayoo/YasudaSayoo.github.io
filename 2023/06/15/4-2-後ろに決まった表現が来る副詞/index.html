4-2-後ろに決まった表現が来る副詞

後ろに決まった表現が来る副詞

言葉と例文

ウォーミングアップ

是不是有点奇怪？——

A:最近、うちの店、お客さんがさっぱり来るんですよ。もう大変でね。
B:えっ！？お客さんは、来るんですか、来ないんですか。

言葉

  • 否定　〜ない／ないだろう
    • 一切〜ない
      • 病気になってから、彼は一切タバコを吸わなくなった。
    • さっぱり〜ない
      • お酒を飲み過ぎて、昨夜のことはさっぱり覚えていない。
    • 少しも・ちょっとも〜ない
      • 靴の汚れがとてもひどくて、洗ってもちっともきれいにならなかった。
    • そう〜ない
      • まだ、そう疲れていないので、もう少し後で休憩をしましょう。
    • 大して〜ない
      • 人気の映画を見に行ったが、大して面白くなかった。
    • 別に〜ない
      • その問題は、普通の問題で、別に難しい問題じゃないよ。
    • とても〜ない／できない／だめだ／無理だ
      • こんなに難しい問題は、私にはとてもとけません。
    • なかなか〜ない／できない
      • 先週引いた風邪が、薬を飲んでいるのに、なかなか治らない。
    • 必ずしも〜ない
      • 先生の言うことが必ずしもすべて正しいわけではない
    • まさか〜ないだろう／まい／思わなかった
      • まさか同じクラスの山田さんと川口さんが結婚するとは思わなかった。
    • めったに〜ない
      • アルバイトばかりしているらしく、彼はめったに学校に来なかった。
  • 推量　〜かもしれない／だろう／らしい
    • ひょっとすると／ひょうっとしたら〜かもしれない
      • 頑張っていたから、ひょっとすると彼は試験に受かるかもしれないよ。
    • もしかすると／もしかしたら〜かもしれない
      • 元気だとは言っているけど、もしかしたら彼は病気かもしれない。
    • おそらく〜だろう／と思う／に違いない
      • この寒さと空の様子からすると、おそらく夕方から雪になるだろう。
    • どうも〜らしい／ようだ
      • 彼の様子が変だ。どうも私に何か隠しているらしい。
  • 様子　〜そうだ／ようだ
    • あたかも〜ようだ／ごとし
      • 田中さんは、あたかも自分が見てきたかのように、その事故の話をした。
    • ちょうど〜ようだ／みたいだ
      • そのシャツの模様は、ちょうど着物の様子のような感じで、とてもきれいだった。
    • まるで〜ようだ／みたいだ
      • もう十月なのに、まるで真夏のように暑い。
  • 仮定　〜ば／たら／なら／ても〜
    • いくら〜ても
      • いくら勉強しても、日本語が上手にならないんですが、どうすればいいでしょう。
    • たとえ〜ても
      • たとえ親が反対しても、私は留学はしたいのだ。
    • 万一／万が一〜ば／たら／なら／だろうか
      • 万が一事故を起こして、人にけがでもさせたら、大変なことになるよ。
  • 疑問　〜か／だろう／だろうか
    • 一体〜疑問詞（なぜ／誰／どこ……）〜か／だろう
      • 来ると言っていたので、彼はまだ来ない。一体いつになったら、来るんだろうか。
    • 果たして〜か／だろうか
      • 彼は大丈夫だと言うが、果たして彼を信じて、お金を貸してもいいのだろうか。

2.基本練習

用法

下線部に注意して、□のなかから適当な言葉を選んで、（　）に入れなさい。

  • ちょうど　　おそらく　　なかなか
    • あそこの岩が（ ちょうど ）ベンチみたいになってますから、一休みしませんか。
    • レストランで食べた料理をまねして作ってるんだけど、（ なかなか ）うまくいかない。
    • 彼は約束の時間通りに来たことがないから、（ おそらく ）今日も遅れてくるだろう。
  • まさか　　たとえ　　果たして
    • （ たとえ ）彼があのとき言った言葉が嘘だったとしても、私は彼を信じます。
    • 選挙でいろいろと約束するのはいいが、（ 果たして ）本当に実行できるのだろうか。
    • 何度もチェックしたから、（ まさか ）ミスがあるとは思わなかった。
  • どうも　　さっぱり　　万が一
    • 火はちゃんと消してね。（ 万が一 ）火事でも起こったら、大変だから。
    • 彼女は（ どうも ）調子が悪いようだ。朝からふらふらしている。
    • 社内では、大ヒット間違いなしと言われていた商品だったが、（ さっぱり ）売れなかった。
  • まるで　　一体　　とても
    • 彼は（ まるで ）映画の中の王子様のようにすてきです。
    • こんなつらい生活が（ 一体 ）いつまで続くのだろう。
    • 彼の言っていることには、（ とても ）賛成できない。

意味

□の中から適当な言葉を選んで、（　）に入れなさい。

  • 別に　　めったに　　大して
    • 彼は（ 大して ）仕事もしていないのに、なぜか出世するのが早い。
    • 東京で四月に雪が降るなんて、（ めったに ）ないことだよ。
    • 試合に負けたのは、（ 別に ）彼がミスしたからじゃないよ。
  • どうも　　ちょうど　　あたかも
    • このスープは（ ちょうど ）日本の味噌汁のようなもので、私の国では毎日飲むんです。
    • 彼は今日はずっとんいこにこしている。（ どうも ）何かいいことでもあったようだ。
    • （ あたかも ）天使が空から降りてきたかのように、空から雪が輝きながら降ってきた。
  • 一切　　必ずしも　　さっぱり
    • 彼ほどひどい人はいないと思ったので、（ 一切 ）付き合わないことにした。
    • 彼らの議論は専門的過ぎて、（ さっぱり ）わからなかった。
    • けんかは始めた方が絶対に悪いとは（ 必ずしも ）言えないと思う。
  • たとえ　　万が一　　いくら
    • （ たとえ ）何が起ころうと、彼女の決意は変わらないだろう。
    • （ いくら ）考えても、答えがわからなった。
    • （ 万が一 ）のことがあるから、必ずシートベルトをしめてください。

用法

下線の言葉を使い方が正しい分には⭕️、間違っている文には❌を（　）に入れなさい。
また、間違っている場合には、下線の言葉に代わる正しい言葉を書きなさい。

  • 今回のテストは、ちょっと難しくなかった。 ( ❌ )  大して 
  • どうぞ道を間違っているような気がする。 ( ❌ )  どうも・何だか 
  • ひょっとすると、彼はここには来ないに違いない。 ( ❌ )  恐らく・多分・きっと 
  • けんかをして出て行って以来、一切連絡がない。 ( ⭕️ )   
  • 外国で生活することは、大して簡単じゃない。 ( ❌ )  そう 
  • あの人は、いくらお酒を飲んでも酔っ払わない。 ( ⭕️ )   
  • そこから見る景色は、まるで夢の中のようだった。 ( ⭕️ )   
  • 場所が悪いのか、必ずしも魚が釣れない。 ( ❌ )  さっぱり・ちっとも 

用法

下線の部分を変えて、正しい文にしなさい。

  • 今日は重要な会議があるから、まさか休むだろう。 → おいしかった 
  • あの映画は、ちっとも面白いです。 → 面白くないです 
  • 彼女はまるで母親らしく、子供をかわいがった。 → 母親のように 
  • 万が一、宝くじに当たると、マンションを買いたい。 → 当たったら 
  • あんなに仲のいい二人がまさかけんかはしない。 → しないだろう・しないと思う 
  • 私の国では、電車で寝てしまう人は、そういます。 → いません 

3.実践練習

1.( )に入れるのに最も良いものを、１・２・３・４から一つ選びなさい。

  • 彼は、知らないことでも、（ ３ ）知っているかのごとく話す。
    • とても
    • ちょうど
    • あたかも
    • おそらく
  • 今、駅に行けば、（ ３ ）彼に会えるかもしれませんよ。
    • まさか
    • まるで
    • もしかすると
    • 万が一
  • 試験があるから、最近は（ ４ ）遊んでいない。
    • 別に
    • 必ずしも
    • とても
    • ちょっとも
  • （ １ ）あの人気アイドルが逮捕されるとは思わなかった。
    • まさか
    • てっきり
    • とても
    • めったに
  • こんなに遅く家を出て、（ ４ ）彼は試験に間に合うのだろうか。
    • まるで
    • どうも
    • ひょっとすると
    • 果たして
  • あの事件の犯人と私は（ ２ ）関係ありません。
    • 少し
    • 一切
    • ちょっと
    • さっぱり
  • 彼は、政治のことなど（ ３ ）知らないのに、偉そうにテレビで解説している。
    • 大そう
    • どうも
    • 大して
    • 大変
  • このパソコン、（ ２ ）キーをたたいても、反応しない。壊れちゃったのかな。
    • たとえ
    • いくら
    • 万一
    • よく
  • 面接試験にTシャッツで来るなんて、（ ３ ）何を考えているんだろうか。
    • 一切
    • 一回
    • 一体
    • 一見
  • 「その缶ジュース飲んでいいよ。」「なんだ、（ ３ ）残ってないじゃないか。」
    • さっぱり
    • 一切
    • ちっとも
    • なかなか

2.__の言葉に意味が最も近いものを、１・２・３・４から一つ選びなさい。

  • このゲーム、難しくて、なかなか先に進めないんだ。（ １ ）
    • 思ったように
    • 思った通り
    • 全然
    • 一切
  • 彼女は、どうも僕を避けているような気がする。（ ３ ）
    • 絶対に
    • どちらかというと
    • なんとなく
    • 間違いなく
  • いくら断っても、彼女は私を旅行に誘ってくるので、困っている。（ ２ ）
    • 何回か
    • どんなに
    • 必死に
    • はっきり
  • 田中さんは、食事もせず、ため息ばかりで、さっぱり元気がない。（ ２ ）
    • ほとんど
    • まったく
    • すっきり
    • なぜか
  • こんな難しい曲は、とても弾けないよ。（ １ ）
    • どうやっても
    • なかなか
    • 何とか
    • すごく

3.次の言葉の使い方として最も良いものを、１・２・３・４から一つ選びなさい。

  • どうも （ ４ ）
    • 試験に合格するために、どうも今から頑張らなければならない。
    • 花瓶を割ったのは、どうもあの子のはずだ。
    • どうもご家族の方にもよろしくお伝えください。。
    • どうも僕の言った言葉が誤解されているような気がする。
  • まさか （ ３ ）
    • まさか彼は来るに違いない。
    • こんなにおいしい料理を作れるなんて、まさかプロだね。
    • まさか彼が医者になるなんて思わなかった。
    • これはまさか君にぴったり仕事だよ。
  • もしかしたら （ １ ）
    • 彼女はもしかしたら昨日徹夜したのかもしれない。
    • あの人は、ここに寄ってから、もしかしたら会社に行くんです。
    • 今夜は、もしかしたら雨が降るに違いない。
    • こんなにも大量のデータを処理するのは、もしかしたら大変なことだ。
  • さっぱり （ １ ）
    • 先生はとても早口で話されたので、さっぱり聞き取れなかった。
    • 久しぶりのお休みだったので、今日は、さっぱり遊んでしまった。
    • こんなに周りがうるさくては、先生の話がさっぱりわかるわけがない。
    • 一人でパーティーに行くのは嫌なので、かれがさっぱり来なかったら、私も行きません。
  • まるで （ ４ ）
    • このゲームは、まるで高いので、あまり売れないだろう。
    • 昨日、遊んだ公園は、まるで広かったけど、ただの草原のようだった。
    • 彼女は、将来、まるで立派な人になるような気がする。
    • 彼は、全然感情を顔に出さない。まるでロボットのようだ。
