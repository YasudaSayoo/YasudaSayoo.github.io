2-5-オノマトペ

オノマトペ

言葉と例文

ウォーミングアップ

下面哪一个是正确的？——

田中さんは（おっとりな　とっとりの　おっとりした）性格で、みんなから好かれている。

言葉

  • ものの様子（〜と）
    • びっしり（と）　彼のノートにはびっしりと数字が書き込まれていた。
    • ぎっしり（と／だ／に／な／の）　箱にはお菓子がぎっしり入っていた。
    • くっきり（と）　今日は天気が良く、富士山がくっきり見えた。
    • とっさり（と）　お土産にミカンをどっさりもらった。
    • びっしょり（と／だ／に／な／の）　一時間走って、びっしょり汗をかいた。
  • ものの様子（〜する／〜している／〜と）
    • ぽかぽか（と／する／している）　薬を飲んだら、体がぽかぽかしてきた。
    • ひんやり（と／する／している）　朝の高原の空気はひんやりとしていた。
    • しっとり（と／する／している）　このクリームを使うと、肌がしっとりする。
    • じめじめ（と／する／している）　雨の日が続き、部屋の中がじめじめしている。
    • ずっしり（と／する／している）　銅でできた花瓶はずっしりと重かった。
    • ひっそり（と／する／している）　放課後の教室はひっそりとして、静かだった。
  • ものの様子（〜だ／〜に／〜な〜／〜の〜）
    • ぐちゃぐちゃ（だ／に／の）　彼女の部屋の中はぐちゃぐちゃだった。
    • めちゃめちゃ（だ／に／の）　台風の後で、町はめちゃめちゃな状態だった。
    • すれすれ（だ／に／の）　海面すれすれのところを鳥が飛んでいた。
    • ほやほや（だ／の）　この店で焼き立てほやほやのパンを食べた。
    • ぼろぼろ（だ／に／の）　十年間履いた靴なので、もうぼろぼろだ。
  • 人の様子（〜と）
    • さっさと　おしゃべりしないで、さっさと食べなさい。
    • せっせと　親鳥はせっせと雛に餌を運んできた。
    • てきぱき（と）　彼女は家事をてきぱきと片付けた。
    • こつこつ（と）　彼はこつこつと受験の準備を進めていた。
    • じっくり（と）　一度自分の将来のことをじっくり考えない。
    • のろのろ（と／する）　のろのろ歩いていると、夕方までに家に帰れないよ。
    • しみじみ（と）　手紙を読み、親の優しさをしみじみ感じた。
    • つくづく（と）　彼女が泣くのを見て、つくづく自分が悪かったと反省した。
    • ざっと　昨日の夜、教科書をざっと読んでおいた。
    • ずばり（と）　自分の欠点をずばりと言われたので、腹が立った。
    • すんなり（と）　彼が社長になることがすんなり決まった。
  • 人の様子（〜する／〜している／〜と）
    • おっとり（と／する／している）　彼はおっとりしていて、あまり怒らない。
    • のんびり（と／する／している）　たまには温泉にでも入ってのんびりしたい。
    • のびのび（と／する／している）　両親が旅行で留守なので、のびのびできる。
    • いきいき（と／する／している）　子供たちの目は生き生きとしていた。
    • きびきび（と／する／している）　彼はきびきびとした動きで荷物を運んだ。
    • いらいら（する／している）　渋滞で車が進まず、いらいらした。
    • むかむか（する／している）　彼の言い訳を聞いていると、むかむかする。
    • きっぱり（と／する／している）　彼はきっぱりとした態度で誘いを断った。
    • はきはき（と／する／している）　面接での回答がはきはきしていて、よかった。
    • おどおど（と／する／している）　面接の時、その学生はおどおどしていた。
    • そわそわ（と／する／している）　姉の結婚式の朝、父はそれそれしていた。
    • くよくよ（と／する／している）　試験に落ちたぐらいで、くよくよするな。
    • にやにや（と／する／している）　彼は私の顔を見てにやにやしていた。
    • ひやひや（する／している）　妹の運転にはいつもひやひやさせられる。
    • ぼんやり（と／する／している）　ぼんやりしていて、パンを買うのを忘れた。
    • ぐったり（と／する／している）　今日は彼は二日酔いでぐったりしていた。
  • 人の様子（〜だ／〜に／〜な〜／〜の〜）
    • くたくた（だ／に／な／の）　6時間も山道を登ってきたので、くたくただ。
    • へとへと（だ／に／な／の）　6時間連続で試験を受けて、もうへとへとだ。

基本練習

連語

例のように一緒に使う言葉を線で結びなさい。

  • びっしょり生える
  • じっくり疲れる
  • びっしり考える
  • ぐったりぬれる
  • ひっそりと冷える
  • しっとりと湿っている
  • ずっしりと静まる
  • ひんやりと思い
  • ずばり育つ
  • ざっと当てる
  • つくづく数える
  • のびのび思う
  • ぽかぽか指示する
  • しみじみ感じる
  • のんびり温まる
  • てきぱき過ごす
  • こつこつ答える
  • はきはき断る
  • きっぱり近く
  • すれすれに続ける
  • よくよく話す
  • くたくたに悩む
  • にやにや疲れる
  • おどおど笑う

意味

□の中から適当な言葉を選んで、（　）に入れなさい。

  • ぎっしり　　くっきり　　どっさり
    • 来月まで予定が（ ぎっしり ）詰まっていて、買い物に行く時間がない。
    • 友達が果物を（ どっさり ）持って、お見舞いに来てくれた。
    • 新しい眼鏡を買ったら、何でも（ くっきり ）と見えるようになった。
  • さっさと　　ざっと　　すんなりと
    • 留学したいと親に言ったら、反対されると思っていたので、（ すんなりと ）認めてくれた。
    • 今日はいつもより遅く起きたので、（ さっさと ）準備をしないと仕事に遅れてしまう。
    • 時間がなくて、報告書を（ ざっと ）読んだだけなので、詳しいことはまだわからない。
  • すれすれ　　ぼろぼろ　　ほやほや
    • 彼は受験勉強の時に使ったという（ ぼろぼろ ）になったノートをを僕に見せてくれた。
    • この道は狭くて、店の前（ すれすれ ）のところを車が通るから危ない。
    • その会社は、出来立て（ ほやほや ）の新しい会社だった。
  • いきいき　　きびきび　　はきはき
    • 夢だった弁護士になって、山本さんの表情は（ いきいき ）している。
    • 面接の時は、（ はきはき ）と大きな声で話したほうがいいよ。
    • 田中さんは（ きびきび ）とした動作で、あっという間に机の上を片付けた。
  • のびのび　　ぼんやり　　のんびり
    • 風邪を引いて、頭が（ ぼんやり ）しているので、何も考えられない。
    • 今週はとても忙しかったから、日曜日は何もせず（ のんびり ）寝ていたい。
    • この辺りは自然が豊かで、子供たちが外で自由に（ のびのび ）遊ぶことができる。
  • ひやひや　　それそれ　　おどおど
    • いつも怒られてばかりなので、先生と話すときは（ おどおど ）した話し方になってしまう。
    • 資格試験の合格発表の日は、朝から（ そわそわ ）して落ち着かなかった。
    • 親に嘘をついていたことがばれるのではないかと（ ひやひや ）した。
  • いらいら　　へとへと　　にやにや
    • いつも私のことをばかにして（ にやにや ）笑うので、私は彼のことが嫌いだ。
    • 娘がまったく言うことを聞かないので、彼女は（ いらいら ）していた。
    • 医者である夫は、毎日夜遅くまで働いて、（ へとへと ）。jになって帰ってくる
  • おっとり　　ぐったり　　くよくよ
    • 兄は、仕事でミスをすると、いつまでも（ くよくよ ）している。
    • 山田さんは性格が（ おっとり ）していて、怒っているところを見たことがない。
    • 彼は、マラソンでゴールした後、その場に倒れこんで（ ぐったり ）として動かなかった。

用法

下線の部分を変えて、正しい文にしなさい。

  • 子供にいたずらをされて、母親はかんかんしていた。 → かんかんだった 
  • 弟が勉強の邪魔をするので、むかむかになった。 → にした 
  • 遠慮しないで、きっぱりに自分の意見を言った。 → きっぱりと・きっぱり 
  • 部屋の中がじめじめと乾いている。 → 湿っている 
  • あの人の言うことはめちゃくちゃしている。 → だ 
  • 子供の頃は、毎日こつこつと遊んだ。 → 勉強した・本を読んだ 
  • 人が少ない、ひっそりの町を歩いた。 → ひっそりとした

3.次の言葉の使い方として最も良いものを、１・２・３・４から一つ選びなさい。

  • ほやほや （ ２ ）
    • 彼に電話をしたら、ほやほやに家まで来てくれた。
    • 新婚ほやほやの川口さんはとても幸せそうだ。
    • やっぱり、ほやほやのご飯はおいしい。
    • 顔ほやほやのところまで近づいて、犬の写真を撮った。
  • むかむか （ ２ ）
    • このレストランの料理はむかむかしすぎていて、おいしくない。
    • わがままな彼の顔を見ただけで、腹が立ってむかむかする。
    • 彼女はとても優しい性格で、いつもむかむかしている。
    • 私が転んだのを見て、彼はむかむか笑っていた。
  • くよくよ （ ４ ）
    • 8時間まずに働いたので、家に帰った時にはくよくよだった。
    • 部屋の中がくよくよしていて、とても汚い。
    • あこがれていた先生とお話しできると聞いて、とてもくよくよした。
    • 終わったことをいつまでもくよくよ考えていても仕方がない。
  • そわそわ （ １ ）
    • 今日は大事なお客様が家に来るので、母は朝からそわそわしている。
    • もう少しで車がぶつかりそうになったので、そわそわした。
    • 誰もいない会議室は、そわそわしていた。
    • 彼の提案は、まったく反対する人がおらず、そわそわと採用された。
  • せっせと （ ４ ）
    • せっせとプレゼントを用意したので、ぜひ受け取ってください。
    • もう8時だから、せっせと準備をして出かけないと。
    • 山田さんが私がやることにせっせと文句を言うので、腹が立つ。
    • 山口さんは、授業料のために、せっせとアルバイトをしてお金をためている。
答え：

  • 「ほやほや」表示“刚刚处于那种状态”。1应该用「すぐ」，3应该用「ほかほか」，4应该用「すれすれ」等。
  • 「むかむか」意为“生气”或“恶心”。
