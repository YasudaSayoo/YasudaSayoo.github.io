6-1-体の言葉を使った慣用表現①

体の言葉を使った慣用表現①

言葉と例文

ウォーミングアップ

下面的说法是什么意思？——

A：これからちょっと課長のところに顔を出してきます。
B：顔を出す？どういう意味ですか。

言葉

  • 顔
    • 〜が広い　彼女は顔が広いので、いろいろな人を紹介してもらうことがある。
    • 〜を貸す　少し頼みがあるので、明日、顔を貸して欲しい。
    • 〜を出す　帰る前に事務所に顔を出して、明日の予定を確認した。
  • 頭
    • 〜が上がらない　とてもお世話になった人なので、頭が上がらない。
    • 〜が痛い　不景気で給料が下がってしまい、来月の生活費のことで頭が痛い。
    • 〜が高い　帰る前に事務所に顔を出して、明日の予定を確認した。
    • 〜に来る　何度やめて欲しいと言っても、やめないので、本当に頭に来る。
    • 〜を下げる　迷惑をかけた人に頭を下げる。
    • 〜を冷やす　頭を冷やしてから、もう一度話し合おうと思う。
  • 目
    • 〜がない　甘いものに目がないので、ケーキを見えるとつい食べてしまう。
    • 〜に浮かぶ　私の合格の知らせを聞いて喜ぶ両親の様子が目に浮かんだ。
    • 〜に付く　家族の写真をいつも目につくところに置いている。
    • 〜を付ける　これから注目の分野としてアニメに目を付けている。
    • 〜を通す　先生がレポートに目に通して、コメントをしてくれた。
  • 口
    • 〜がうまい　店員があまりにも口がうまいので、つい買いすぎてしまった。
    • 〜が重い　彼女は家庭がうまくいっていないらしく、家族の話になると口が重くなる。
    • 〜が堅い　彼女は口が堅いから、絶対に誰にも話さないだろう。
    • 〜が軽い　彼は口が軽いから、あまり重要なことは話さないほうがいい。
    • 〜が滑る　うっかり口滑って、言ってはいけないことを話してしまった。
    • 〜に合う　彼のために料理を作ったが、彼の口に合うだろうか。
    • 〜にする　初めは不満を口にしていたが、今はあきらめて何も言わなくなった。
    • 〜に出す　何度も口に出して練習すると、覚えられる。
    • 〜を出す　自分に関係のないことに口を出すのは、あまりよくないと思う。
  • 耳
    • 〜が痛い　練習が足りないことは自分でもよくわかっていて、監督の言葉は耳が痛い。
    • 〜が早い　さっき他の人に話したことを知っているとは、彼はなんて耳が早いんだ。
    • 〜にする　この店はとてもおいしいという話を耳にしたので、食べに来た。
    • 〜を貸す　経験者の意見には耳を貸したほうがいい。
  • 鼻
    • 〜が高い　みんなが兄のことを褒めるので、妹として鼻が高い。
    • 〜で笑う　一生懸命考えて出したアイデアを鼻で笑われて無視されたので、頭に来た。
  • 手
    • 〜が掛かる　私は、わがままで、手が掛かる子供だったそうで、親は大変だったらしい。
    • 〜が空く　その作業が終わって手が空いたら、こちらも手伝ってください。
    • 〜が足りない　今日はアルバイトの人が急に休んだので手が足りなくて、困った。
    • 〜が出ない　学生の私には、五万円のバッグは高すぎて手が出ない。
    • 〜が離せない　今、忙しくて手が離せないので、一時間後に来てください。
    • 〜につかない　合格発表の前日は、結果が気になって何も手につかなかった。
    • 〜を貸す／借りる　重い荷物を運ぶのに弟が手を貸してくれた。
    • 〜を組む　これからはライバル会社と手を組んで海外に進出する。
    • 〜を出す　いろいろな事業に手を出した結果、会社の経営が傾いた。／先に手を出したほうが悪いと思うが、その原因を作ったのは相手のほうだ。
    • 〜をつける　宿題が多すぎて、何から手をつけていいのかわからない。
  • 腕
    • 〜がいい　あの歯医者は腕がいいという評判だ。
    • 〜を上げる／が上がる　一年前と比べて料理の腕を上げた。
  • 肩
    • 〜を落とす　「計画が失敗するとは思わなかった」と肩を落として、話していた。
    • 〜を貸す　足が痛くて歩けなかったので、肩を貸してもらって部屋まで戻った。

2.基本練習

連語

例のように一緒に使う言葉を線で結びなさい。

  • 口がうまい自分の弱みについて言われて、聞くのが辛い
  • 目がない技術などがこれまでより上達する
  • 耳が痛い口だけでほめたり、人の気分をよくするように話す
  • 口が軽い自分の気持ちがコントロールできないくらい好きである
  • 腕を上げるおしゃべりで、言ってはいけないことも話してしまう
  • 目を通すとても世話が必要である
  • 手が掛かる自慢ができる
  • 鼻が高い状況が想像できる
  • 目に浮かぶ全部を大体見ること
  • 耳にする聞く・聞こえてくる
  • 手をつける怒る・怒りの気持ちを持つ
  • 頭が痛い自慢ができる
  • 鼻が高い状況が想像できる
  • 目に浮かぶ全部を大体見ること
  • 耳にする聞く・聞こえてくる
  • 口が堅い人の言うことを聞く
  • 耳を貸すばかにする
  • 鼻で笑う秘密など言ってはいけないことを簡単に話さない
  • 頭が固いあまり話さない・おしゃべりではない
  • 口が重い新しい考えなどをなかなか理解や納得しようとしない
  • 顔が広い知人が多い
  • 口が滑るある場所にいく
  • 手がでない冷静になる・気持ちを落ち着かせる
  • 頭を冷やす自分の能力ではできない
  • 顔を出す人に話してはいけないことをうっかり話してしまう

用法

下線の言葉を使い方が正しい分には⭕️、間違っている文には❌を（　）に入れなさい。
また、間違っている場合には、下線の言葉に代わる正しい言葉を書きなさい。

  • 前よりもずっと料理の腕がよくなった。 ( ❌ )  腕があがった 
  • あまりにも頭に来たので、思わず手をつけてしまった。 ( ❌ )  手を出して 
  • あまり好きではない人だと、悪いところが目を付ける。 ( ❌ )  目に付く 
  • けがをした友人に肩を貸して、部屋まで連れていった。 ( ⭕️ )   
  • 会議にちょっと顔を出して、すぐに出かけて。 ( ⭕️ )   
  • 発表資料を作ると時、友人が顔を貸してくれた。 ( ❌ )  手を貸して 
  • 彼女は私が失敗した様子を見て、目で笑った。 ( ❌ )  鼻で笑った 
  • 口が堅い人だと思っていたら、実はとても話好きだった。 ( ❌ )  思い 

用法

□の中から適当な言葉を選んで、（　）に入れなさい。

  • 耳　　手　　口　　目
    • この間あった彼女のことが気になって何も（ 手 ）につかない。
    • まだ無名の画家の絵に（ 目 ）を付けた。
    • そのうわさは以前（ 耳 ）にしたことがある。
    • 自分の気持ちを正直に（ 口 ）に出すことはなかなかできない。
  • 手　　鼻　　口　　顔
    • 祖父は社長をしていたので、とても（ 顔 ）が広い。
    • つい（ 口 ）が滑って、友達の秘密を話してしまった。
    • 会場の準備をしているんだけど、（ 手 ）が足りないので、何人か手伝ってください。
    • 自分の子供が優秀な成績を取って（ 鼻 ）が高い。
  • 頭　　口　　手　　肩
    • 不合格の知らせに（ 肩 ）を落とした。
    • 思ったことをすぐ（ 口 ）にしないほうがいい。
    • この問題は難しすぎて（ 手 ）が出ない。
    • 彼は妻には（ 頭 ）が上がらない。

3.実践練習

1.＿＿の言葉に意味が最も近いものを、１・２・３・４から一つ選びなさい。

  • 興奮しているので、少し落ち着いたほうがいいと思う。（ ３ ）
    • 口にした
    • 耳を貸した
    • 頭を冷やした
    • 肩を落とした
  • 合格の知らせを聞いて、彼女が喜ぶ顔が想像できる。（ ４ ）。
    • 目に付く
    • 目に遭う
    • 目に来る
    • 目に浮かぶ
  • 彼はおしゃべりで、言ってはいけないこともすぐにぺらぺらと話してしまう。（ ２ ）。
    • 口が滑って
    • 口が軽くて
    • 口を出して
    • 口がうまくて
  • 彼女は自分の意見を聞いてもらえなくて、がっかりしていた。（ ４ ）。
    • 手につかないで
    • 頭が下がって
    • 耳が痛くなって
    • 肩を落として
  • まず自分のできることから始めるつもりだ。（ ２ ）。
    • 手を貸す
    • 手をつける
    • 腕を組んで
    • 腕を貸す

2.__の言葉に意味が最も近いものを、１・２・３・４から一つ選びなさい。

  • 組む　（ ３ ）
    • 子供の手を組んで、歩いている。
    • いろいろな考えをよく組んで報告書を書いた。
    • 二人で手を組んで、計画を立てた。
    • この計画はよく組んであるので、成功するだろう。
  • 空く　（ ４ ）
    • 腕が空いたので、少し休むことにした。
    • 仕事が早く終わって、やっと頭が空いた。
    • 朝からが目が空かなくて、新聞を読む暇もない。
    • 今、手が空かないので、ちょっと待ってください。
  • 通す　（ ２ ）
    • 友人を心を通すことができて、うれしい。
    • 試験前に教科書に目を通す。
    • 大阪まで行ったので、少し足を通す京都まで行った。
    • 親のアドバイスに耳を通したほうがいいと思う。
