単語の前に漢字をプラス

言葉と例文

ウォーミングアップ

(1) 你知道带有「未・不・無・非」？——

(2) 加上了这些汉字，词义有什么变化？——

言葉

    • 生なま〜　生放送・生演奏・生出演

基本練習

用法

□の中から適当な漢字を選んで、（　）に入れなさい。

  • 非　　末　　不　　無
    • 日本に十年も住んでいると、日本の習慣が（ 無 ）意識のうちに身につく。
    • 時間がなかったので、（ 未 ）完成のままレポートを提出してしまった。
    • アルバイトの面接に一時間も遅れていくなんで、（ 非 ）常識ですよ。
    • このドラマの舞台は日本なのに、登場人物がみんな英語で話すなんで（ 不 ）自然だ。
  • 総　　全
    • 東京都の（ 総 ）面積は約2,187平方キロメートルだ。
    • 彼は火災で（ 全 ）財産を失った。
  • 各　　諸
    • 緊急の場合は、（ 各 ）家庭に学校から直接連絡します。
    • この本は、日本と（ 諸 ）外国との経済的な関係についての本である。
  • 現　　今
    • （ 今 ）シーズンは何回スキーに行けるかなあ。
    • この研究が成功するかどうか、（ 現 ）階段では予測できない。
  • 前　　元
    • 彼女のお母さんは（ 元 ）歌手だそうだが、彼女はそんなに歌がうまくない。
    • 今年度は（ 前 ）年度と比べて売り上げが伸びない。
  • 真　　正
    • 彼女はその会社に（ 正 ）社員として採用された。
    • 山田さんの（ 真 ）後ろに座っている人が田中さんですよ。
  • 逆　　反
    • 駅は（ 反 ）方向ですよ。そっちじゃなくて、こっちです。
    • この池の周りを（ 逆 ）時計回りに三周回ってください。
  • 別　　異
    • 友達と一緒に旅行に行ったが、目的が違うので、ほとんど（ 別 ）行動だった。
    • 留学は（ 異 ）。

意味

対になるよう□の中から漢字を選んび、（　）に入れなさい。二回使う漢字もあります。

  • 長　　好　　短　　片　　半　　悪　　両
    • （ 好 ）条件　——　（ 悪 ）条件
    • （ 長 ）袖　——　（ 半 ）袖
    • （ 長 ）期間　——　（ 短 ）期間
    • （ 両 ）足　——　（ 片 ）足
  • 大　　重　　低　　新　　高　　旧　　軽　　少　　多
    • （ 大・多 ）人数　——　（ 少 ）人数
    • （ 高 ）カロリ　——　（ 低 ）カロリ
    • （ 新 ）体制　——　（ 旧 ）体制
    • （ 重 ）工業　——　（ 軽 ）工業

意味

下線の言葉に意味が最も近いものを線で結びなさい。

  • 超高級ホテルに泊まる録画や録音をしていない
  • ピアノの生演奏を聞きながら食事する一番・最も
  • これはジャズの名曲だ優れた・有名な
  • 化粧品の主成分を調べる中心となる・主な
  • 環境問題が今後の最重要課題だ非常に・とても

用法

下線の言葉に意味が最も近いものを線で結びなさい。

  • 小川選手は全国大会で良成績を収めた。 ( ❌ )  好 
  • この薬はよく効くが、副作用が心配だ。 ( ⭕️ )   
  • この店は質の高い商品を安価格で販売している。 ( ❌ )  低 
  • 使わないお皿を旧新聞で包んでしまっておこう。 ( ❌ )  古 
  • この用紙に、氏名と今住所を記入してください。 ( ❌ )  現 
  • A社は軽自動車では日本で一番有名だ。 ( ⭕️ )   
  • A国はこの十年で経済的に速成長している。 ( ❌ )  急 
  • 彼は若いころ少太りだったが、今はスマートだ。 ( ❌ )  小 
  • この高校の生徒の多部分は大学に進学する。 ( ❌ )  大 
  • 私の無注意で、息子に大けがをさせてしまった。 ( ❌ )  不
